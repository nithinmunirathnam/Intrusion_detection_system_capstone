#Smart Intrusion Detection System (Capstone Project)

A Python-based smart security solution that detects intrusions using motion sensors and facial recognition, and sends alert notifications with captured images.

🧠 Features
- Human detection through motion sensors
- Facial recognition using OpenCV & LBPH
- Real-time alerting via email
- Data logging and reporting for each event

🚀 Tech Stack
- Python
- OpenCV
- LBPH Face Recognition
- SMTP Email Alerts

📊 Skills Practiced
- Data Processing
- Data Handling
- Real-time data workflows
- Reporting
- Python scripting
